# find-peaks
find optional number of peaks( return x and y val of each peaks of any spectrum data files)

spectrum data format should be like

|:---|:---:|---:|
|1 |2 |3 |
|4 |5 |6 |

| Pixel | Intensity|
|:---|:---:|---:|
| 0 | 1 |
| 1 | 13 |
| 2 | 30 |
| 3 | 43 |
| 4 | 31 |
| 5 | 11 |
| ... |

Curent supectrum data files supoort:
None

.asc .csv .xlsx .xls .pickle .txt .html(table)
